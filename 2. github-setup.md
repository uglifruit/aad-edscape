# 1. Install the GitHub CLI

## Mac:
```
brew install gh
```
## Windows:
```
winget install --id GitHub.cli
```
## Linux (Debian/Ubuntu):
```
sudo apt install gh
```
# 2. Authenticate

Run:
```
gh auth login
```
You'll get an interactive prompt:
1.	Where do you use GitHub? → GitHub.com
2.	Preferred protocol? → HTTPS (recommended for most people)
3.	Authenticate Git with GitHub credentials? → Yes
4.	How would you like to authenticate? → Login with a web browser

It will then show you a one-time code (e.g. A1B2-C3D4) and open your browser. Paste the code into the browser, authorise the app, and you're done.

# 3. Verify it worked
```
gh auth status
```
You should see something like:
github.com
  ✓ Logged in to github.com account YourUsername
  - Active account: true
  - Git operations protocol: https
